# Brave Backup Tool v2.1.1

## Highlights

- **Fix:** log messages no longer show `{0}`, `{1}` placeholders — copy progress displays real file names, counts, and paths
- **Multi-profile Brave:** detect all profiles from `Local State` (e.g. Default, Profile 1)
- **Profile picker (GUI):** choose which profiles to back up or restore (checkboxes, select all / none)
- **Profile picker (console):** pick profiles by number or `A` for all
- **Restore:** `Local State` and root files copied only when restoring all profiles from a backup
- Stronger Brave process shutdown before backup/restore

## Downloads

- **BraveBackup.exe** — Windows, no install
- **docs/POMOC-pl.md** — help (Polish)
- **docs/HELP-en.md** — help (English)

## Requirements

- Windows 10/11
- Brave Browser installed (for profile data path)
