# Brave Backup Tool v2.1.0

## Highlights

- Polish and English UI (PL / EN) with language toggle and settings
- First-run onboarding (3 steps)
- Improved settings dialog (readable options, backup count 5/10/15/20)
- Security info window, status bar, empty state on backup list
- Safer restore flow (confirmation + optional pre-restore backup)
- Delete confirmation shows the actual backup folder name
- Single-file EXE build (i18n + GUI bundled)

## Downloads

- **BraveBackup.exe** — Windows, no install
- **docs/POMOC-pl.md** — help (Polish)
- **docs/HELP-en.md** — help (English)

## Requirements

- Windows 10/11
- Brave Browser installed (for profile data path)
