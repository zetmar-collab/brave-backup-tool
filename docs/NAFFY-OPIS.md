# Opis produktu — naffy.io

## Tytuł
**Brave Backup Tool** — kopia zapasowa profilu Brave na Windows

## Krótki opis (1–2 zdania)
Darmowe narzędzie na Windows do tworzenia i przywracania kopii zapasowej profilu przeglądarki Brave: zakładki, hasła, historia, rozszerzenia i sesje kart. Jeden plik EXE, bez instalacji, z prostym interfejsem graficznym.

## Opis rozszerzony

Brave Backup Tool to lekka aplikacja dla użytkowników **Brave Browser** na Windows, którzy chcą zabezpieczyć swój profil przed awarią dysku, aktualizacją systemu lub przeniesieniem na nowy komputer.

Program uruchamiasz jednym plikiem **BraveBackup.exe** — bez instalatora i bez konta. Przed kopią prosi o zamknięcie Brave i zamyka wszystkie procesy przeglądarki, żeby pliki (m.in. hasła i ciasteczka) zostały skopiowane poprawnie.

### Co potrafi
- **Pełna kopia** profilu (bez folderów cache — szybsza i mniejsza niż ręczne kopiowanie całego katalogu)
- **Kopia kluczowych danych** — zakładki, loginy, historia, rozszerzenia, sesje kart, ustawienia
- **Przywracanie** z wybranej kopii (nadpisanie lub pełne przywrócenie profilu)
- **Własny folder kopii** — wybierasz dysk i ścieżkę (np. dysk zewnętrzny)
- **Lista kopii** z datą i rozmiarem, odświeżanie, usuwanie starych
- **Automatyczna rotacja** — przechowuje do 10 ostatnich kopii

### Dla kogo
- Użytkownicy Brave na Windows 10/11
- Osoby robiące backup przed formatowaniem PC
- Każdy, kto chce prostego narzędzia zamiast ręcznego kopiowania `%LOCALAPPDATA%\BraveSoftware\...`

### Wymagania
- Windows 10 lub 11
- .NET Framework (standardowo w systemie)
- Zainstalowana przeglądarka Brave
- PowerShell nie jest wymagany do codziennego użytku (tylko do budowy EXE ze źródeł)

### Bezpieczeństwo
Kopie zawierają dane logowania i ciasteczka. Przechowuj je w bezpiecznym miejscu i nie udostępniaj innym osobom.
